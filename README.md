# Inventory Management API

This project is a Flask-based REST API for managing inventory items.

## Features

- View all inventory items
- View one item by ID
- Add new item
- Update item
- Delete item
- Fetch external product data using barcode
- CLI interaction

## Technologies Used

- Python
- Flask
- Requests

## How to Run

1. Run the Flask app:

python3 app.py

2. Open browser or Postman:

http://127.0.0.1:5000/inventory

3. Run CLI tool:

python3 cli.py
